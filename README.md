# eco-map-generator

Tooling for generating maps for [Eco](https://play.eco/).

## About Eco

Eco is a multiplayer survival/simulation game by Strange Loop Games. Players collaborate to build a civilization on a shared procedurally-generated planet and stop an incoming meteor, while an ecological simulation tracks the damage their extraction, pollution, and land use do to the biosphere.

It features:

- A real in-game government — players write and vote on laws.
- A player-driven economy with contracts and currency.
- A skill/profession tree that forces specialization, which is why mods like Global Planetary Defense (spreading endgame crafting across many specializations) and the BunWulf mods (new professions, recipes, farming) exist.
